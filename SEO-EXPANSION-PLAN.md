# SEO Expansion Plan - Mallorca Reformer

## 10 additional improvements to maximize SEO traffic
1. Build full locale versions (`/en`, `/de`, `/es`, `/sv`) for all high-intent pages, not only landing pages.
2. Add neighborhood landing pages (e.g., Santa Catalina, Portixol, Palmanova) with unique local content.
3. Create programmatic pages for intent modifiers: beginner, private, rehab-safe, prenatal, athlete.
4. Deploy internal link modules by intent: every blog links to 1 directory page, 1 comparison page, 1 signup CTA.
5. Add `FAQPage` schema blocks to all core transactional pages and high-traffic blogs.
6. Add `Review` + `AggregateRating` schema where verified studio reviews are available.
7. Create comparison clusters: Reformer vs Mat Pilates, Reformer vs Yoga, Private vs Group, Pilates vs strength training.
8. Publish data-backed local market reports quarterly (pricing, availability, trend pages) to attract backlinks.
9. Build partner pages for hotels/retreats in Mallorca that offer Reformer classes and link to studio pages.
10. Launch digital PR + link outreach with local tourism/wellness sites and expat publications.
